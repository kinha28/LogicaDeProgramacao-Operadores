# 💻 Operadores Lógicos e Ternários em JavaScript

Exercício criado para praticar operadores relacionais, lógicos e ternários utilizando JavaScript.

## 🚀 Conteúdos praticados

- Variáveis
- `prompt()`
- `alert()`
- Conversão de tipos
- Operadores relacionais
- Operadores lógicos
- Operador ternário
- Valores booleanos (`true` e `false`)
- Lógica de programação

## 📂 Exercício

O programa:
- recebe dados do usuário
- verifica condições utilizando operadores relacionais
- utiliza operadores lógicos (`&&`, `||`, `!`)
- utiliza operador ternário para decisões
- exibe resultados na tela

## 🛠️ Tecnologias utilizadas

- HTML5
- JavaScript

## 🎯 Objetivo

Praticar lógica de programação e fortalecer os fundamentos do JavaScript para desenvolvimento front-end.
